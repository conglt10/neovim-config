# 1. Requirements

## Ripgrep

```bash
sudo apt-get install ripgrep
```

## Ast-grep

```bash
cargo install ast-grep
```

## fd-find

```bash
sudo apt update
sudo apt install fd-find
```

## Nerd font

### 1.Download font

https://www.nerdfonts.com/font-downloads

### 2. unzip

```bash
unzip [file name downloaded]
```

### 3. Create folder fonts

```bash
mkdir -p ~/.fonts
```

### 4. Copy file fonts

Copy all `.ttf` to fonts folder `~/.fonts`

### 5.

```bash
fc-cache -fv
```
